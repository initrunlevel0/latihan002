Folder untuk report homework Map Reduce menggunakan MRJOB module running dalam Python 2.7
yang berisi file : 

	README, sebagai kontent introductory dari isi semua file dalam folder ini
	4 File kode MapReduce in Python in .py extension
	4 File hasil running kode MapReduce in .txt extension
	
Kode MapReduce yang di submit di jelaskan sebagai berikut : 

	1. program : tunggulsm_artname_count.py, adalah program yang menghitung berapa banyak nama seorang artis muncul dalam listening habits user lastfm. Output dari program ini adalah file tunggulsm_artname_count.txt berukuran 5 MB.
	
	2. program : tunggulsm_user_with_favorit_song_count.py, adalah program yang menghitung berapa banyak seorang user lastfm memainkan sebuah lagu. Output dari program ini adalah file tunggulsm_user_with_favorit_song_count.txt berukuran 346 MB.
	
	3. program : tunggulsm_song_by_artist_count.py, adalah program yang menghitung berapa banyak seorang artist memainkan sebuah lagu. Output dari program ini adalah file tunggulsm_song_by_artist_count.txt berukuran 127 MB.
	
	4. program : tunggulsm_trackname_count.py, adalah program yang menghitung berapa banyak sebuah track lagu dimainkan oleh user lastfm. Output dari program ini adalah file tunggulsm_trackname_count.txt berukuran 30 MB.
