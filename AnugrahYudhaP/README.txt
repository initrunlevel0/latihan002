 10 file kode program untuk tugas MapReduce
 10 file hasil output program untuk tugas MapReduce

10 pasang file kode program dan hasil output program untuk tugas MapReduce tersebut adalah :
1. usercounter.py melakukan penghitungan jumlah aktivitas mendengarkan lagu yang tercatat dalam sistem untuk masing-masing user.
   Hasil output program usercounter.py adalah usercounter.txt

2. bandsummary.py melakukan penghitungan jumlah aktivitas mendengarkan lagu yang tercatat dalam sistem berdasarkan band/artist yang didengarkan.
   Hasil output program bandsummary.py adalah bandsummary.txt

3. songsummary.py melakukan penghitungan jumlah aktivitas mendengarkan lagu yang tercatat dalam sistem berdasarkan judul lagu yang didengarkan.
   Hasil output program songsummary.py adalah songsummary.txt

4. songplayedbyyear.py melakukan penghitungan jumlah aktivitas mendengarkan lagu dalam 1 tahun tertentu yang tercatat dalam sistem selama periode tersebut.
   Hasil output program songplayedbyyear.py adalah songplayedbyyear.txt

5. songplayedbyyearmonth.py melakukan penghitungan jumlah aktivitas mendengarkan lagu dalam 1 bulan tertentu yang tercatat dalam sistem selama periode tersebut.
   Hasil output program songplayedbyyearmonth.py adalah songplayedbyyearmonth.txt

6. songbandlist.py melakukan penghitungan jumlah aktivitas mendengarkan lagu yang tercatat dalam sistem berdasarkan judul lagu oleh band/artist yang menyanyikannya.
   Hasil output program songbandlist.py adalah songbandlist.txt
   File songbandlist.txt dihapus karena berukuran lebih besar dari 50 MiB (60 MiB)

7. songbandsummary.py melakukan rekapitulasi terhadap aktivitas mendengarkan lagu yang tercatat dalam sistem berdasarkan judul lagu oleh band/artist yang unik/berbeda.
   Input dari program songbandsummary.py adalah songbandlist.txt dan hasil output program songbandsummary.py adalah songbandsummary.txt

8. usersongbandlist.py melakukan penghitungan jumlah aktivitas mendengarkan lagu oleh user yang tercatat dalam sistem berdasarkan judul lagu oleh band/artist yang menyanyikannya.
   Hasil output program usersongbandlist.py adalah usersongbandlist.txt
   File usersongbandlist.txt dihapus karena berukuran lebih besar dari 100 MiB (324 MiB)

9. usersongbandsummary.py melakukan rekapitulasi terhadap aktivitas mendengarkan lagu oleh user yang berbeda/unik yang tercatat dalam sistem berdasarkan judul lagu oleh band/artist yang unik/berbeda.
   Input dari program usersongbandsummary.py adalah usersongbandlist.txt dan hasil output program usersongbandsummary.py adalah usersongbandsummary.txt

10. usercountrysummary.py melakukan penghitungan jumlah user yang tercatat dalam sistem berdasarkan negara asal user tersebut.
    Input dari program usercountrysummary.py adalah userid-profile.tsv dan hasil output program usercountrysummary.py adalah usercountrysummary.txt
