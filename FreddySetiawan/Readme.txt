Ini merupakan Tugas Mengolah Dataset dengan MapReduce:
Dataset yang diolah adalah: http://www.dtic.upf.edu/~ocelma/MusicRecommendationDataset/lastfm-1K.html

Kolom meliputi: userid, timestamp, artist_id, artist_name, track_id, track_song_name

Analisa pertama:
1. track_counter.py
	Program Python untuk mencari tahu berapa kali suatu lagu/track diputar.
2. output_track_counter.txt
	Hasil Output Program track_counter.py

Analisa kedua:
1. artis_counter.py
	Program Python untuk mencari tahu berapa kali seorang artis dimainkan musik-musiknya (Group By Artist Name).
2.	output_artis_counter.txt
	Hasil Output Program track_counter.py
