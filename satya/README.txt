Berikut ini adalah penjelasan file-file yang ada di folder ini:

1. playedbyuser.py adalah script python untuk melihat jumlah lagu yang diputar oleh setiap lagu 
dari data userid-timestamp-artid-artname-traid-traname.tsv
2. playedbyuser.txt adalah hasil run python script playedbyuser.py

3. genderuser.py adalah script python untk melihat seberapa banyak pria dan wanita yang menggunakan lastfm
dari data userid-profile.tsv
4. genderuser.txt adalah hasil run python script genderuser.py

5. userage.py adalah script pyhton untuk melihat sebaran usia pengguna lastfm 
dari data userid-profile.tsv
6. userage.txt adalah hasil run python script userage.txt

7. songidplayed.py adalah script python untk melihat seberapa kali satu lagu diputar menggunakan song id 
dari data userid-timestamp-artid-artname-traid-traname.tsv
8. songidplayed.txt adalah hasil run python script songidplayed.py
